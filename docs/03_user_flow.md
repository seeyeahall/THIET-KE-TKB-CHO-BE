# User Flow

## Luong khoi tao

1. Phu huynh dang ky/dang nhap.
2. Tao ho so be dau tien.
3. Chon avatar, tuoi, so thich, mau sac, con vat yeu thich.
4. Cau hinh AI provider neu chua co.
5. Tao lich tuan dau tien bang thu cong hoac AI.
6. Be vao trang chu va bat dau dung.

## Luong be hang ngay

1. Be chon avatar cua minh.
2. Be thay chu de hom nay va lich hom nay.
3. Be chon hoat dong dang den gio.
4. Be xem huong dan bang anh/card ngan.
5. Be bam hoan thanh.
6. Be nhan xu, XP hoac huy hieu.
7. AI goi y hoat dong tiep theo dua tren context.

## Luong tao lich thu cong

1. Be/phu huynh mo man hinh thiet ke lich.
2. Chon ngay.
3. Keo hoat dong tu thu vien vao khung gio.
4. Sua thoi gian/thoi luong.
5. Luu lich.

## Luong AI tao lich

1. Be/phu huynh bam "Tao giup con tuan moi".
2. Backend nap child profile, so thich, lich hien tai, lich su hoat dong.
3. AI tao lich dang JSON theo schema.
4. Backend validate ket qua.
5. Frontend hien preview.
6. Phu huynh/be chap nhan hoac sua.
7. Luu vao database.

## Luong quan ly AI provider

1. Admin/phu huynh nhap provider, endpoint, API key, model.
2. Backend gui request test.
3. Neu thanh cong, provider co the duoc kich hoat.
4. Neu that bai, khong cho dat lam mac dinh.

## Luong storage media

1. Upload avatar/anh hoat dong len Supabase Storage.
2. Luu URL va metadata vao database.
3. Frontend lay anh qua public URL hoac signed URL.
4. Cloudflare cache anh public de tang toc.
