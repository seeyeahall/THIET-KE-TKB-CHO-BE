# Admin Panel

## Vai tro

Admin panel danh cho phu huynh va admin he thong. Day la noi quan ly du lieu va cau hinh, khong phai trai nghiem chinh cua be.

## Khu vuc phu huynh

- Quan ly ho so be.
- Xem lich ngay/tuan.
- Them/sua hoat dong rieng cho gia dinh.
- Xem lich su hoat dong.
- Xem xu, XP, huy hieu.
- Cau hinh gioi han va ghi chu an toan.

## Khu vuc AI

- Them provider.
- Test provider.
- Chon provider mac dinh.
- Bat/tat capability: chat, image, tts, stt.
- Xem log loi gan day.

## Khu vuc activity library

- Tao activity.
- Sua activity.
- Duyet activity do AI tao.
- An/hien activity.
- Upload anh minh hoa.

## Khu vuc storage

- Xem media assets.
- Upload avatar/anh hoat dong.
- Xoa media khong dung.
- Gan media vao activity/theme/avatar.

## Khu vuc system

- Cau hinh domain backend.
- Cau hinh Cloudflare cache.
- Cau hinh Supabase project.
- Export/import seed data.

## Nguyen tac UI admin

Admin co the dense hon UI cua be, nhung van can gon:

- Table cho danh sach.
- Form ngan.
- Badge trang thai.
- Preview anh.
- Confirm truoc khi xoa.
